How it works

Skills for AI agents

01
Install once

One command installs the skill into your agent's context. Works with Claude Code, Cursor, Codex, and any SKILL.md-compatible agent.

02
Agent loads on demand

The skill uses progressive disclosure — SKILL.md loads first, then specific reference files are loaded only when needed. Keeps context lean.

03
Write better code

Your agent now knows correct method signatures, return types, SEBI rules, and TradeHull's coding style — without you explaining it every time.

04
Stays updated

Run npx skills update when a new version ships. Skills are versioned — you control when to update.
